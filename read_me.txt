Program.sh - Основной файл для запуска скрипта. Выбор пунктра происходит по средствам ввода определенной цифры в командную строку.
0 -- возврат к предыдущему действию.

В первой части можно выполнить:
1) поиск студентов не сдахвщих хотя бы один тест;
2) вывод списка группы, упорядоченного по кол-ву попыток сдачи теста.

В первой части можно выполнить:
1) вывод средней оценки выбранного студента;
2) вывод занятий с максимальной посещаемостью для выбранной группы;
3) вывод занятий с минимальной посещаемостью для выбранной группы.
